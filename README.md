# Lab-2
Dataset Summary
The dataset used in this project is the Iris dataset from Kaggle.
It contains measurements of iris flowers including sepal length, sepal width, petal length, and petal width. The dataset is commonly used for machine learning classification tasks.
Machine Learning Problem
This project aims to classify iris flowers into different species based on their physical measurements.
The target variable is the species of the flower. This is a supervised classification machine learning problem.
